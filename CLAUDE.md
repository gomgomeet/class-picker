# class-picker

수강신청/학급 도구 웹앱 프로젝트. 튜토리얼 문서와 발표 자료(PPT)를 함께 만드는 중이다.

## 규칙

- 사용자에게는 항상 한국어로, 초보자 눈높이로 설명한다.
- **중요한 작업 하나가 끝날 때마다, 그리고 사용자가 "저장", "백업", "오늘 끝" 이라고 말하면 반드시 `/save-work` 스킬을 실행한다** (PROGRESS.md 기록 + 커밋 + 푸시).
- 새 세션을 시작하면 먼저 `PROGRESS.md`를 읽고 지난 작업과 "다음에 할 일"을 파악한 뒤 이어서 작업한다.
- 튜토리얼은 `tutorial.md`, 발표 자료 초안은 `slides.md`에 저장한다.
- 대화 기록은 컴퓨터마다 따로 저장되므로, 작업 내용은 반드시 파일로 남겨 push해야 다른 곳에서 이어서 할 수 있다.
